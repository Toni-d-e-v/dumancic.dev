## Overview

TAPP is a modern SaaS platform that enables cafes, restaurants, hotels, and bars to digitize their menus and implement contactless ordering systems. Guests simply scan a QR code from their table to instantly view the complete menu on their phone in their preferred language and place orders directly — no printing, no contact, no app required. Orders arrive instantly to the venue's tablet/laptop with sound notifications, streamlining operations and improving guest experience.

## Key Features

### Ordering System
- **Table-Side Ordering** — guests scan QR code and automatically see their table number
- **Real-Time Notifications** — instant order alerts with sound notifications on tablet/laptop
- **Waiter Call Function** — guests can directly call waiter for service or bill
- **Custom Orders** — easy addition of sides, special requests, and notes
- **Multi-Language Ordering** — tourists order in their native language without confusion
- **Order Management Dashboard** — track all incoming orders in real-time
- **No Errors** — digital orders eliminate miscommunication and wrong orders

### Digital Menu
- **QR Code Menu** — digital menu accessible instantly after scanning
- **Landing Page Builder** — customizable page with all links (Instagram, Google Maps, reservations)
- **Instant Updates** — price and item changes go live immediately, no reprinting needed
- **Custom Branding** — logo upload, custom colors, removable TAPP branding
- **Image Uploads** — high-quality photos for menu items increase sales by 40%+
- **Mobile-First Design** — optimized for smartphone viewing experience

### Analytics & Insights
- **Live Analytics** — tracking scan counts, popular items, peak hours
- **Order Analytics** — most ordered dishes, revenue tracking, performance metrics
- **Real-Time Dashboard** — monitor everything as it happens
- **Revenue Reports** — comprehensive sales data and trends

- TypeScript
- Tailwind CSS + Radix UI
- TanStack Query (React Query)
- React Hook Form + Zod validation
- QRCode.react for QR generation

- WebSocket for real-time notifications

**Backend:**
- Node.js + Express.js
- TypeScript
- MongoDB + Mongoose
- Socket.io for real-time order updates
- JWT authentication + bcrypt
- AWS S3 for image storage
- Sharp for image optimization
- Express Validator
- Push notification system

**DevOps & Tools:**
